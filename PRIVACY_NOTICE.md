# ScanMate AI Pro Privacy Notice Placeholder

ScanMate AI Pro is designed as an offline-safe document scanner.

## Local-first behavior

- Scans, imported images, PDFs, OCR text files, QR images and ZIP backups are saved in app-managed storage.
- The app does not require login.
- The app does not require a custom backend.
- The app does not include hardcoded API keys.

## Optional AI behavior

AI tools are optional. If the user adds their own Gemini API key and presses Generate, the prompt text is sent to Google's Gemini API. If no key is configured, AI requests are disabled.

## Sharing

When the user taps Share, Android's system share sheet is used. The selected app controls what happens after the user chooses a destination.
